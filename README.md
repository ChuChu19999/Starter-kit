### Миграции базы данных

```bash
alembic revision --autogenerate -m "migration"
```
**Описание:** Создает новую миграцию на основе изменений в моделях SQLAlchemy. Команда автоматически анализирует текущие модели и сравнивает их с состоянием базы данных, генерируя SQL скрипт для применения изменений. Параметр `-m` задает сообщение для миграции.

```bash
alembic upgrade head
```
**Описание:** Применяет все непримененные миграции к базе данных, доводя схему до актуального состояния (до версии "head"). Выполняет SQL скрипты миграций последовательно, обновляя структуру таблиц, индексов и ограничений.
